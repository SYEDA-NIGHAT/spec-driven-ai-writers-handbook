# Chapter 3 — Deploying the Book

Once your book is ready:

### Step 1 — Create GitHub repo
Make a public GitHub repository.

### Step 2 — Update config
Set your username + repo name in `docusaurus.config.js`.

### Step 3 — Deploy
Run:

```bash
npm run deploy
